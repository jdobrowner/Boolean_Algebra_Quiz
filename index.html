So you think you're good at logic? Let's find out.

Before taking this quiz you should be familar with logical equivalences, truth tables, and the conditional operator. The syntax is that of most programming languages.

During the quiz you will have access to a table of logical equivalences and a “scratch paper” text area in which you can work out a problem, as shown below.

logical equivalences-

!(p && q) = !p || !q De Morgan's laws
!(p || q) = !p && !q
p || (p && q) = p Absorption laws
p && (p || q) = p
p || q = q || p Commutative laws
p && q = q && p
(p || q) || r = p || (q || r) Associative laws
(p && q) && r = p && (q && r)
p || (q && r) = (p || q) && (p || r) Distributive laws
p && (q || r) = (p && q) || (p && r)
p || p = p Idempotent laws
p && p = p
p && true = p Identity laws
p || false = p
p || true = true Domination laws
p && false = false
p || !p = true Negation laws
